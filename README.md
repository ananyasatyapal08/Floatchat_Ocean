# Floatchat Ocean 

**Floatchat Ocean** is a simple web-based chat dashboard project built using HTML, CSS and JavaScript. It provides a visually ocean-themed interface for signing in, chatting, and navigating features in a dashboard layout.

##  Features

- Ocean-themed UI with custom styling (see `ocean.png`, `bg.png`, etc).  
- Sign-in page for user login (static/placeholder for now).  
- Dashboard layout after login: navigation, chat view, etc.  
- Chat interface: users can exchange messages in a chat page.  
- Features page to highlight what the app offers.  
- About page to provide details or credits.

##  Purpose and Use-cases

This project serves as:

- A prototype or concept of a chat dashboard with a themed UI.  
- A learning project for front-end skills: HTML, CSS, JavaScript.  
- A base you can expand into a full real-time chat app (e.g., adding backend, database, websockets).  
- A visual portfolio piece (especially useful for your upcoming studies/tech journey).
